How to publish a new post:

  1. Fill in the form below and click Generate JSON.
  2. Copy the generated JSON entry.
  3. Open posts.json in your code editor.
  4. Paste the new entry at the top of the array (after the opening [), followed by a comma.
  5. Save the file and push to GitHub — your post is live!
